# 🎨 Random Color Generator

A simple web project that generates a random color and shows its HEX code.

## Features
- Generates a random color
- Displays the HEX color code
- Built with simple HTML, CSS, and JavaScript

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository:
git clone https://github.com/GE160/random-color-generator.git

2. Open the project folder.

3. Open `index.html` in your browser.
