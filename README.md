# 🎨 Random Color Generator

A simple and elegant web application that generates random colors with a valid HEX code! Perfect for designers, developers, and anyone who needs some colorful inspiration!

## ✨ Features

- 🎯 Generates random colors with one singular single click!
- 🔤 Displays the HEX color code for easy copying, sharing and identification
- 🎨 Visual color display with nice, smooth transitions
- 📋 supports Copying to your clipboard!
- 💾 Clean, simple and intuitive user interface

## 🛠️ Used languages

- **HTML5** - Structure and markup
- **CSS3** - Styling and animations
- **JavaScript** - Color generation and interactivity

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/v33550336/random-color-generator.git
   ```

2. Navigate to the project folder:
   ```bash
   cd random-color-generator
   ```

3. Open `index.html` in your browser

## 🚀 Usage

1. Open the application in your web browser
2. Click the "Generate Color" button to create a random color
3. The HEX code will be displayed and updated automatically
4. Click on the HEX code to copy it to your clipboard
